## ПОЛНЫЙ КАТАЛОГ МЕТОДОВ АРБИТРАЖА

### КАТЕГОРИЯ A: ЦЕНОВОЙ АРБИТРАЖ

**A1. Cross-Exchange Spot Arbitrage**
Купить на бирже A дёшево, перевести, продать на B дорого.
- Доходность: 0.1-0.5% | Частота: 5-50/день | Риск: средний
- Вердикт: НЕ РЕКОМЕНДУЕТСЯ — трансферный риск 5-30 мин

**A2. Cross-Exchange Futures Arbitrage (ЕСТЬ)**
LONG perp + SHORT perp на разных биржах.
- Доходность: 0.05-0.30% | Частота: 2-20/день | Риск: низкий
- Уже реализовано. Улучшения: maker-taker, снизить min_spread до 0.15%

**A3. Triangular Arbitrage (ПРИОРИТЕТ #1)**
Цепочка 3 пар на ОДНОЙ бирже: USDT->BTC->ETH->USDT
- Доходность: 0.02-0.15% за цикл | Частота: 50-500/день | Риск: очень низкий
- Мин. капитал: $50 | Сложность: средняя

Почему лучшая для нас:
1. ОДНА биржа — весь капитал $21 в одном месте
2. 3 ордера за менее 100ms — минимальный риск
3. Не нужны фьючерсы — только спот
4. Высокая частота — много мелких прибылей
5. Maker ордера экономят комиссию

Математика: edge 0.05%, maker fees 0.06% (3 ноги по 0.02%), net +0.02% = $0.004 за цикл. 50 сделок/день = $0.20/день = $6/мес.

**A4. Statistical Arbitrage (Pairs Trading)**
Торгуем корреляцию (BTC/ETH, SOL/AVAX). z-score > 2 = SHORT, < -2 = LONG.
- Доходность: 0.1-0.5% | Частота: 5-20/день | Риск: средний
- Вердикт: рекомендуется как дополнение

**A5. Cross-Exchange Spot Pre-deposited**
Спот-арбитраж без переводов (балансы заранее на обеих биржах).
- Доходность: 0.05-0.20% | Частота: 10-50/день
- Вердикт: можно добавить минимальными изменениями

### КАТЕГОРИЯ B: FUNDING и CARRY

**B1. Funding Rate Arbitrage (ПРИОРИТЕТ #2, код есть)**
LONG на бирже с низким funding + SHORT на бирже с высоким.
- Доходность: 10-40% APR | Частота: 3 расчёта/день | Риск: низкий
- Код в funding_arbitrage.py ГОТОВ, нужно подключить к engine.py
- Пример: BTC diff 0.02% per 8h, $500 = $0.10/день, $3/мес

**B2. Cash and Carry (ЕСТЬ)**
BUY spot + SHORT perp на одной бирже.
- Доходность: 5-30% APR | Риск: очень низкий
- Реализовано в cash_and_carry.py

**B3. Funding Rate Harvesting (ПРИОРИТЕТ #3)**
Агрессивный вход за 1-2 часа до funding, выход сразу после.
- Доходность: 15-60% APR в бычьем рынке | Мин. капитал: $20
- Отличие от Cash-and-Carry: короткие входы (2-3 часа вместо дней)
- В бычьем рынке funding часто 0.1-0.3% per 8h = 100%+ APR

**B4. Perpetual-Quarterly Basis Trade**
SHORT quarterly + LONG perp -> базис сужается к expiry.
- Доходность: 5-20% APR | Риск: очень низкий
- Вердикт: позже при капитале >$500

### КАТЕГОРИЯ C: СТРУКТУРНЫЙ АРБИТРАЖ

**C1. DEX-CEX Arbitrage (ПРИОРИТЕТ #5, будущее)**
Арбитраж между Uniswap/Jupiter/PancakeSwap и Bybit/OKX.
- Доходность: 0.5-5% за сделку | Частота: 10-100/день
- Мин. капитал: $200+ | Сложность: высокая (нужен Web3)
- DEX цены обновляются медленнее -> спреды 1-10% на мем-коинах
- Оптимальные сети: Solana ($0.01 gas), BSC ($0.05), Arbitrum ($0.10)
- Реализация: Фаза 1 мониторинг DexScreener API -> Фаза 2 хедж на CEX -> Фаза 3 свопы на DEX

**C2. Listing Arbitrage**
Покупка при листинге на новой бирже, продажа где уже торгуется.
- Доходность: 5-50% | Частота: 1-5/неделю | Риск: высокий
- Мониторинг: Binance/OKX/Bybit announcements API

**C3. Launchpool/Airdrop Arbitrage**
Стейкинг в Launchpool + хедж шортом = бесплатные токены.
- Доходность: 10-100% APR на залоченный капитал

### КАТЕГОРИЯ D: HIGH-FREQUENCY

**D1. Market Making** — НЕ для нас (требует колокацию)

**D2. Orderbook Imbalance Trading (ПРИОРИТЕТ #8)**
Предсказание движения по дисбалансу bid/ask в ордербуке.
- Доходность: 0.02-0.10% | Частота: 50-200/день
- У нас УЖЕ есть WS orderbook feeds и depth data
- Нужно: imbalance calculator + entry/exit logic

**D3. Latency Arbitrage** — НЕ для нас (нужен VPS рядом с биржами)

### КАТЕГОРИЯ E: EVENT-DRIVEN

**E1. Liquidation Cascade Trading**
Покупка после каскадных ликвидаций (flash crash recovery).
- Доходность: 1-10% за событие | Частота: 1-5/неделю
- Мониторим open interest drops >5% за 5 мин -> LONG

**E2. News Arbitrage** — НЕ для нас (требует NLP)

## ПЛАН ВНЕДРЕНИЯ ПО ФАЗАМ

### ФАЗА 1: НЕМЕДЛЕННО (1-2 недели, $0 доп. вложений)

**1.1 Интеграция Funding Arbitrage в Engine (2-3 дня)**
- Подключить FundingArbitrageStrategy в engine.py build_strategies()
- Добавить config params в TradingSystemConfig
- Включить в ENABLED_STRATEGIES
- Ожидаемый эффект: +$0.05-0.15/день

**1.2 Funding Rate Harvesting (1-2 дня)**
- Модифицировать CashAndCarryStrategy: timing-aware вход за 1-2 часа до расчёта
- Добавить predicted funding rate мониторинг
- Быстрый выход после получения funding
- Ожидаемый эффект: +$0.03-0.10/день

**1.3 Включить Maker-Taker execution (1 день)**
- EXEC_USE_MAKER_TAKER=true — уже поддерживается в коде
- Снижение комиссий на 50-60%
- Ожидаемый эффект: удвоение прибыльных возможностей

### ФАЗА 2: БЛИЖАЙШИЕ (2-4 недели)

**2.1 Triangular Arbitrage Strategy (5-7 дней)**
Новая стратегия — новый файл strategies/triangular_arbitrage.py:
- Сканер треугольных путей для всех доступных спот-пар
- Расчёт implicit cross-rate vs actual rate
- Atomic 3-leg execution (3 ордера за один цикл)
- Maker order support для 2 из 3 ног
- Cooldown per triangle + min profit threshold

Архитектура:
```
TriangularScanner
  -> find_triangles(exchange, all_pairs) -> List[Triangle]
  -> calculate_profit(triangle, orderbooks) -> float
  -> execute_triangle(triangle) -> ExecutionReport
```

Пары для сканирования: BTC/USDT, ETH/USDT, ETH/BTC, SOL/USDT, SOL/BTC, SOL/ETH, BNB/USDT, BNB/BTC, XRP/USDT, XRP/BTC, DOGE/USDT, и тд.

**2.2 Statistical Arbitrage / Pairs Trading (3-5 дней)**
Новый файл strategies/pairs_trading.py:
- Калибровка z-score на исторических данных (rolling window 24-72h)
- Entry: z > 2.0 или z < -2.0
- Exit: z возвращается к 0 +/- 0.5
- Пары: BTC/ETH, SOL/AVAX, DOGE/SHIB, BNB/OKB

### ФАЗА 3: СРЕДНЕСРОЧНЫЕ (1-2 месяца)

**3.1 DEX-CEX Мониторинг (3-5 дней)**
- Интеграция DexScreener API для мониторинга DEX цен
- Алерты когда DEX-CEX спред > threshold
- Пока только сигналы, без автоматической торговли

**3.2 DEX-CEX Хедж через CEX (5-7 дней)**
- При обнаружении спреда: шорт на CEX
- Ручная покупка на DEX или полуавтоматическая

**3.3 Listing Monitor (2-3 дня)**
- Мониторинг Binance/OKX/Bybit announcements
- Telegram алерт при новом листинге
- Авто-подготовка ордеров

### ФАЗА 4: ДОЛГОСРОЧНЫЕ (3-6 месяцев)

**4.1 Полная DEX интеграция**
- Web3 wallet integration (Solana, BSC, Arbitrum)
- Автоматические свопы на DEX
- MEV protection (private transactions)

**4.2 ML-предсказание спредов**
- Features: orderbook imbalance, funding, volatility, time of day
- Модель: gradient boosting -> улучшение тайминга на 20-30%

## ГЛАВНЫЙ ВЫВОД

Для текущего капитала ($21) **Triangular Arbitrage** — самая перспективная новая стратегия:
- Весь капитал на одной бирже (не нужно разделять)
- Высокая частота (50-500 сделок/день)
- Очень низкий риск (одна биржа, 3 ордера за <100ms)
- Работает на споте (без фьючерсных комиссий/funding)

Для наращивания пассивного дохода — **Funding Arbitrage** (код готов!) + **Funding Harvesting**. Это стабильные 10-60% APR при минимальном риске.

Для серьёзного масштабирования ($500+) — **DEX-CEX Arbitrage** даёт самые большие спреды (0.5-5%), но требует Web3 инфраструктуру.
