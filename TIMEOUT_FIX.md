# Timeout Bug Fix - 2026-04-02

## Проблема

Система сообщала о провалах сделок с ошибкой "order_timeout", при этом ни первая, ни вторая нога не заполнялись.

## Анализ логов

### Сделка INJUSDT (2026-04-01 20:47)
```
20:47:34,753 - [EXECUTE] INJUSDT notional=6.51
20:47:38,627 - [MAKER_FILLED] sell INJUSDT on bybit — успешно
20:47:43,554 - [SECOND_LEG_NOT_FILLED] INJUSDT on htx — провал второй ноги
20:47:44,820 - execution_result {'success': False, 'message': 'second_leg_failed'}
```
**Статус**: Это было со СТАРЫМ кодом (до внедрения retry-логики)

### Сделки RESOLVUSDT (2026-04-02 19:51, 20:23, 20:55) - ПОСЛЕ моих изменений
```
19:51:00 - [EXECUTE] RESOLVUSDT
19:51:07 - [MAKER_RETRY] buy on okx attempt=1/3
19:51:10 - execution_reject {'leg': 0, 'reason': 'order_timeout'}
         - execution_latency: 9556ms (9.5 секунд)
```
**Все 3 попытки**: Провал на первой ноге из-за timeout!

## Корневая причина

Обнаружен баг в расчете timeout для первой ноги:

```python
# БЫЛО (неправильно):
first_leg_timeout_sec = self.config.order_timeout_ms / 1000 * 2  # 6 секунд

# Но maker-taker механизм требует:
# - Попытка 1 (maker): 2s
# - Попытка 2 (maker): 2s
# - Попытка 3 (maker): 2s
# - Откат на taker IOC: 3s
# - Накладные расходы: ~3s
# ИТОГО: 12 секунд нужно!
```

**Результат**: Timeout срабатывал РАНЬШЕ, чем maker-taker механизм успевал завершиться!

## Исправление

```python
# СТАЛО (правильно):
first_leg_timeout_sec = self.config.order_timeout_ms / 1000 * 4  # 12 секунд
```

Теперь timeout учитывает полный цикл maker-taker:
- 4 × 3000ms = 12 секунд
- Достаточно для 3 попыток maker + fallback на taker + overhead

## Важные замечания

1. **Retry-логика второй ноги НЕ была протестирована** потому что все сделки проваливались на первой ноге
2. После перезапуска бота с исправленным кодом, первая нога должна успевать заполняться
3. Только тогда можно будет протестировать retry-логику второй ноги

## Статус retry-логики второй ноги

Код реализован правильно (lines 224-291 в execution.py):

- ✅ Escalation на market order на 3-й попытке
