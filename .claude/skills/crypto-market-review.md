# Crypto Market Analysis Review

You are acting as:

• a professional crypto trader  
• a quantitative analyst  
• a senior software engineer

Your task is to review a function that evaluates the situation on the cryptocurrency market.

The function may include:

- indicators
- trend analysis
- market sentiment
- volatility analysis
- decision logic

## Step 1 — Understand the function

Explain:

- what the function does
- what inputs it expects
- how the market evaluation is produced

## Step 2 — Trading logic review

Evaluate the algorithm from a trading perspective.

Check:

• whether the market logic makes sense  
• whether the indicators are used correctly  
• whether the conclusions about market state are justified  

Identify:

- incorrect assumptions
- weak trading logic
- missing indicators

## Step 3 — Engineering review

Evaluate the implementation:

• code clarity  
• algorithm structure  
• data usage  
• edge cases  

Check whether the function:

- is deterministic
- handles missing data
- handles extreme market conditions

## Step 4 — Signal quality

Evaluate whether the algorithm could produce:

• false signals  
• delayed signals  
• biased results  

Explain why.

## Step 5 — Improvements

Suggest:

- algorithm improvements
- additional indicators
- structural improvements in the code

## Output format

### Function summary

### Trading logic evaluation

### Engineering evaluation

### Weaknesses

### Suggested improvements
