#### 2. Фильтр по лотности и минимальной цене
**Проблема**: Бот пытается торговать ВСЕМИ акциями из списка, даже если на них не хватает денег.

**Решение**: Добавить предварительную фильтрацию при старте.

**Файл для изменения**: `stocks/system/factory.py` или `stocks/system/engine.py`

```python
async def filter_affordable_tickers(
    provider, tickers: List[str], max_price: float = 100.0
) -> List[str]:
    """Фильтрует тикеры по доступности: цена * lot_size <= max_affordable."""
    affordable = []
    for ticker in tickers:
        try:
            snap = await provider.get_snapshot(ticker)
            cost_per_lot = snap.quote.last * snap.lot_size
            if cost_per_lot > 0 and cost_per_lot <= max_price:
                affordable.append(ticker)
        except Exception:
            pass
    return affordable
```

**Добавить в `cb_stock_start` (handlers/stock_handlers.py:182)**:
```python
# После build_stock_engine
config.tickers = await filter_affordable_tickers(provider, config.tickers, max_price=350)
logger.info("Filtered to %d affordable tickers: %s", len(config.tickers), config.tickers)
```

#### 3. Комиссии БКС не учтены в расчете P&L
**Проблема**: Комиссия БКС ~0.05% за сделку (~0.1% на round-trip) не вычитается из прибыли.

**Решение**: Добавить учет комиссий в `stocks/system/execution.py`.

**Файл**: `stocks/system/execution.py`

```python
# В методе execute_entry и execute_exit добавить:
BCS_COMMISSION_PCT = 0.0005  # 0.05% за сделку

# При расчете PnL:
gross_pnl = (exit_price - entry_price) * quantity * lot_size
commission = (entry_price + exit_price) * quantity * lot_size * BCS_COMMISSION_PCT
net_pnl = gross_pnl - commission
```

def get_adaptive_rsi_oversold(ticker_volatility: float) -> float:

## РИСКИ И ОГРАНИЧЕНИЯ

### ⚠️ Критические риски

1. **Малый капитал (700₽)**
   - Комиссии БКС съедают ~0.35₽ на сделку (0.05%)
   - При позиции 350₽ это 0.1% - нужен edge > 0.2% для прибыли
   - **Митигация**: Торговать только сигналы с edge > 0.3%

2. **Низкая ликвидность дешевых акций**
   - FEES, OGKB, MSNG - малые объемы торгов
   - Риск проскальзывания > 2%
   - **Митигация**: Фильтр по среднедневному объему > 1 млн ₽

3. **Волатильность энергетических акций**
   - TGKA, HYDR, MRKY - могут резко двигаться ±5-10% за день
   - **Митигация**: Stop-loss 4%, trailing stop 2%

4. **Лимиты БКС API**
   - Обычно 10-50 запросов/сек
   - При 35 тикерах каждые 5 сек = 7 req/sec (в норме)
   - **Митигация**: Ротация тикеров, кэширование

### 📊 Реалистичные ожидания

- **Первый месяц**: +0-10% (50-70₽)
- **Первые 3 месяца**: +20-50% (140-350₽)
- **Первый год**: +50-100% (350-700₽)

**Не ожидайте**:
- ❌ Удвоения капитала за неделю
- ❌ Win rate > 80% (реально 50-60%)
- ❌ Стабильной прибыли каждый день
