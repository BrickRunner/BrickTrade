## Как работает

### Overheat Detector (7 сигналов)
Бот сканирует все 95 пар по 7 критериям:
1. **RSI** > порог (перекупленность)
2. **Price Spike** (резкий рост цены)
3. **Volume Spike** (всплеск объёма)
4. **Funding Rate** > 0 (высокая ставка фондирования)
5. **Open Interest Growth** (рост открытого интереса)
6. **EMA Deviation** (отклонение от скользящей средней)
7. **Divergence** (дивергенция RSI и цены)

### Scoring
- Score: **0-7** (количество активных сигналов)
- **Score ≥ 5** → OVERHEATED (рекомендация на SHORT)
- Автоматическая сортировка по score

### Автоматическая торговля
- **Auto-execute**: открывает SHORT при score ≥ min_score
- **Leverage**: настраиваемое плечо (1-20x)
- **Stop-Loss / Take-Profit**: автоматические
- **Max positions**: лимит одновременных сделок

## Настройки

### Торговые параметры
```env
# handlers/short_handlers.py → EXEC_CONFIG
order_size_usdt = 5.5       # размер позиции в USDT
leverage = 5                 # плечо
max_positions = 3            # макс одновременных шортов
sl_pct = 2.0                 # stop-loss %
tp_pct = 3.0                 # take-profit %
min_score = 5                # минимальный score для входа
auto_execute = True          # авто-вход без подтверждения
```

### Таймфреймы
- **1m**: быстрое скальпирование
- **5m**: сбалансированная торговля (по умолчанию)
- **15m**: среднесрочные позиции

## Риски и управление

### ⚠️ Важно!
- **Высокая волатильность**: мем-коины могут двигаться на ±20-50% за минуты
- **Ликвидность**: проверяйте объём перед входом
- **Плечо**: используйте разумное плечо (2-5x рекомендуется)
- **Stop-Loss**: ОБЯЗАТЕЛЬНО устанавливайте SL

### Рекомендации
1. **Начните с малого**: размер позиции 5-10 USDT
2. **Тестируйте**: запустите авто-скан без торговли
3. **Отслеживайте**: следите за открытыми позициями
4. **Ограничьте риск**: max_positions = 2-3
5. **Min score**: используйте 5-6 для консервативной торговли
